# MIYAR Version History

## MIYAR 2.0 (Current Stable)
**Status:** Completed
**Focus:** Design Intelligence & Automated Evaluation

### Core Capabilities:
- **Intelligent Project Intake (Phase 10A):** Multi-modal project creation via Chat, URL scraping, and manual file uploads. AI-driven parsing of project context, typology, scale, and budget.
- **MIYAR Scoring Engine:** Deterministic 5-dimension scoring framework evaluating Strategic Alignment (SA), Financial Feasibility (FF), Market Positioning (MP), Differentiation Strength (DS), and Execution Risk (ER).
- **Automated Explainability:** Full narrative traces for every score, including score contribution waterfalls, sensitivity analysis toggles, and penalty/risk flag detection.
- **Market Benchmarking:** Live comparative analysis against UAE market data for construction costs, fit-out budgets, and material constants.
- **AI Design Advisor:** Gemini-powered recommendations for space planning, material selection, and styling, guided by dynamic UAE design trends.
- **Investor & Board-Ready Outputs:** Instant generation of formal, watermarked PDF/DOCX design briefs and investor summaries directly from the platform.
- **Public Share Views:** Token-gated, read-only links for sharing project insights with external stakeholders without requiring authentication.
- **Cognitive Bias Alerts (V11):** Autonomous detection of common real estate investment biases (e.g., Optimism Bias, Anchoring) with actionable mitigation steps.
- **Assets Management:** Centralized repository for project source materials, with side-by-side original content viewers and structured JSON data extraction.
- **Dashboard Analytics:** High-level pipeline overview, risk distribution, and quick access to recent projects and platform alerts.

## MIYAR 1.0 (Legacy)
**Status:** Deprecated
**Focus:** Basic feasibility and static reporting.
- Initial proof-of-concept for real estate design evaluation.
- Manual data entry without autonomous multi-modal ingestion.
- Basic cost estimation without advanced explainability traces or bias detection.
